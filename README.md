# kur

Based package manager.
Installs whatever is listed in a `kurfile`.

```
# Common Packages
entr       #brew #ubuntu
gh         #brew #ubuntu
ripgrep    #brew #ubuntu

# Ubuntu Packages
build-essential    #ubuntu

# Brew Packages
bazelisk    #brew
elixir      #brew
fd          #brew
go          #brew
jj          #brew
tree        #brew

# Cargo Packages
bob-nvim          #cargo
cargo-outdated    #cargo
cargo-watch       #cargo
mdbook            #cargo
starship          #cargo
```

Supported platforms are

- Cargo
- OSX/Homebrew
- Ubuntu
- Alpine
- Pip

## Installation

Run this

```
cargo install kur
```
